ℹ️ Información de Configuración
Eventos Webhook Configurados:
  • ONCRMDEALADD - Se crea una nueva oportunidad en Bitrix24
  • ONCRMDEALUPDATE - Se actualiza una oportunidad existente
  • ONCRMDEALDEL - Se elimina una oportunidad en Bitrix24
URL del Webhook:
{{ request.build_absolute_uri }}/app/bitrix/webhook/
Comandos de Consola:

Sincronizar eliminaciones:

python manage.py sync_bitrix_deletions

Configurar webhooks:

python manage.py setup_bitrix_webhooks --webhook-url https://tudominio.com/app/bitrix/webhook/ --bitrix-webhook https://tubitrix.bitrix24.mx/rest/123/abc123/

Sincronización completa (recomendado):

docker-compose exec web python manage.py full_bitrix_sync

Solo detectar perdidas:

docker-compose exec web python manage.py sync_lost_opportunities

Solo importar nuevas:

docker-compose exec web python manage.py import_bitrix_opportunities
